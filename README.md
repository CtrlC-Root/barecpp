# barecpp

BARE C++ Serialization Library

## Quick Start

Update Git submodules:

```bash
git submodules update --recursive --init
```

Configure the build:

```bash
./waf configure
```

Build:

```bash
./waf build
```

Run tests:

```bash
./build/pkg/test/test
```
